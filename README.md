# Deloitte Data Analytics Virtual Experience Program

Completed the Deloitte Data Analytics Virtual Experience Program offered through Forage.

This virtual experience provided hands-on exposure to real-world data analytics tasks using Tableau and Excel to solve business problems and derive actionable insights.

## Task 1: Factory Downtime Analysis (Tableau)

### Objective
Analyze telemetry data collected by Daikibo's manufacturing facilities and identify downtime patterns across factories and machine types.

### Activities Performed
- Imported and processed JSON telemetry data in Tableau.
- Created a calculated field (`Unhealthy`) to quantify potential downtime.
- Built a **Downtime per Factory** bar chart.
- Built a **Downtime per Device Type** bar chart.
- Developed an interactive dashboard with filtering capabilities.
- Identified the factory with the highest downtime and analyzed the affected device types.

### Tools Used
- Tableau Desktop
- Data Visualization
- Dashboard Design

### Key Insight
The analysis revealed that **Daikibo Factory Seiko** experienced the highest downtime, primarily due to issues associated with **LaserWelder** devices.

## Task 2: Equality Score Classification (Excel)

### Objective
Classify employee equality scores across factories and job roles to identify fairness and potential discrimination patterns.

### Activities Performed
- Added a new **Equality Class** column to the dataset.
- Categorized equality scores into:
  - **Fair:** −10 to +10
  - **Unfair:** between 10 and 20 (absolute value)
  - **Highly Discriminative:** greater than 20 (absolute value)
- Applied Excel formulas using `IF` and `ABS` functions to automate the classification process.

### Tools Used
- Microsoft Excel
- Logical Functions (`IF`, `ABS`)
- Data Classification
